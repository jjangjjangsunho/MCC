# ☕ MCC Java Study Record

> 명지대학교 컴퓨터공학과 프로그래밍 동아리 MCC(Myongji Computer Club) 자바 스터디 및 알고리즘 학습 기록 공간입니다.

<br>

## 🎯 학습 목표 (Learning Goals)
오직 **Java** 언어 하나에 집중하여, 백엔드 개발의 뼈대가 되는 핵심 원리와 코딩 역량을 확실하게 다집니다.
* **자바 기본기 마스터:** 변수, 제어문부터 객체지향 프로그래밍(OOP)의 4대 특징까지 완벽한 이해
* **효율적인 데이터 다루기:** 자바 컬렉션 프레임워크(Collection Framework)의 적절한 활용
* **깊이 있는 자바 지식:** JVM의 동작 원리, 메모리 구조, 예외 처리 등 심화 개념 학습
* **문제 해결 능력 향상:** 자바를 활용한 알고리즘 문제 풀이 및 논리적 사고력 기르기
